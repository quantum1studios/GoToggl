# GoToggl

This plugin allows you to track your Toggl time directly from the Godot interface! 

## Requirements
- Godot 3+
- togglkey.json file

### togglkey.json
```
{
    "api_token": "dfd35235dsfsd24523523",
    "workspace_id": 11000245,
    "project_id": 11100111,
    "description": "GoToggl Entry"
}
```
